# AGENTS.md

## Reglas para agentes IA

1. Leer primero:

   - AI_CONTEXT.md
   - CURRENT_TASK.md
   - PROJECT_PLAN.md

2. No modificar archivos fuera de los permitidos en CURRENT_TASK.md.

3. Antes de cambiar código, explicar brevemente el plan.

4. Hacer cambios pequeños.

5. No crear arquitectura innecesaria.

6. No agregar dependencias nuevas sin justificar.

7. Mantener compatibilidad con Python 3.12.

8. El proyecto debe funcionar primero en terminal.

9. No implementar frontend si la tarea actual es backend.

10. No borrar archivos data/\*.json sin autorización.

## Estilo de código

- Código simple.
- Funciones pequeñas.
- Errores claros.
- JSON legible.
- Sin magia innecesaria.
